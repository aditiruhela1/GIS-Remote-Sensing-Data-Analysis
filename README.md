# 🌍 GIS & Remote Sensing Analysis Projects

This repository contains two geospatial analysis projects completed during my GIS & Remote Sensing Internship. The projects focus on applying GIS and Remote Sensing techniques to analyze environmental and urban challenges using satellite imagery and spatial data.

## 📌 Projects

### 1. Drought Severity Mapping using Vegetation Condition Index (VCI)

**Objective:**
Analyze vegetation health using the Vegetation Condition Index (VCI) to identify and map drought-affected areas.

**Key Tasks:**
- Processed satellite imagery
- Generated VCI maps
- Classified drought severity
- Visualized drought-prone regions using GIS

---

### 2. Urban Heat Island Mapping using Land Surface Temperature, Built-up Density, and GIS Analysis

**Objective:**
Study the Urban Heat Island (UHI) effect by analyzing Land Surface Temperature (LST), built-up density, and spatial patterns.

**Key Tasks:**
- Estimated Land Surface Temperature (LST)
- Analyzed built-up density
- Performed spatial analysis
- Generated Urban Heat Island maps
